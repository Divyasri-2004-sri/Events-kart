Thing 1 — Setup Google Sheet (5 minutes)
1. Open this link
👉 https://sheets.google.com → Sign in with your Gmail
2. Create new sheet
Click the big + (Blank spreadsheet)
3. Type these 5 headers in Row 1
Click cell A1 and type:

A1 → Timestamp
B1 → FirstName
C1 → Email
D1 → Phone
E1 → Message

4. Open the script editor
Click Extensions (top menu) → click Apps Script
5. Clear and paste
Delete everything you see in the editor, then paste this:
javascriptfunction doGet(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  sheet.appendRow([
    e.parameter.timestamp,
    e.parameter.firstName,
    e.parameter.email,
    e.parameter.phone,
    e.parameter.message
  ]);
  return ContentService.createTextOutput('OK');
}
6. Save it
Press Ctrl + S → name it ContactForm → click OK
7. Deploy it

Click Deploy (top right) → New deployment
Click the ⚙️ icon → select Web app
Execute as → Me
Who has access → Anyone
Click Deploy → Authorize access → choose your Gmail → click Allow
A URL appears — copy it (starts with https://script.google.com/macros/s/...)


Thing 2 — Paste URL in contact.html
Open your contact.html file in Notepad or any editor.
Press Ctrl + F and search for:
PASTE_YOUR_GOOGLE_SCRIPT_URL_HERE
Replace just that text with the URL you copied. Save the file.
